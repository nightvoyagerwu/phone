# 2024-2025年手机对比工具

这是一个完整的手机对比工具，包含92款主流手机厂商机型的详细参数数据。

## 项目结构

```
phone-comparison-tool/
├── index.html              # 主页面文件
├── style.css               # 样式文件
├── script.js               # JavaScript功能文件
├── data/
│   └── all_phones_unified.json  # 手机数据文件
└── README.md               # 使用说明（本文件）
```

## 使用方法

### 方法一：本地直接打开（推荐）

1. **下载所有文件**：确保下载整个 `phone-comparison-tool` 文件夹
2. **直接打开**：双击 `index.html` 文件在浏览器中打开
3. **如果遇到问题**：请使用本地服务器方式（见下方）

### 方法二：本地服务器（如果直接打开有问题）

1. **启动服务器**：
   ```bash
   # 在项目目录下运行
   python3 -m http.server 8080
   # 或者
   npx serve .
   ```

2. **访问网站**：在浏览器中打开 `http://localhost:8080`

## 部署到自己的网站

1. **上传文件**：将整个 `phone-comparison-tool` 文件夹上传到你的网站服务器
2. **访问地址**：通过 `https://yourdomain.com/phone-comparison-tool/` 访问
3. **集成到现有网站**：可以将HTML内容嵌入到你的网站中

## 故障排除

### 如果数据无法加载：
1. 检查 `data/all_phones_unified.json` 文件是否存在
2. 确保文件路径正确
3. 尝试使用本地服务器方式

### 如果功能无法使用：
1. 检查浏览器控制台是否有错误信息
2. 确保JavaScript已启用
3. 尝试刷新页面

## 技术说明

- **纯静态网站**：无需服务器端支持
- **本地存储**：用户添加的数据保存在浏览器本地
- **响应式设计**：支持手机、平板、电脑访问
- **无依赖**：不依赖任何外部框架

## 更新数据

要更新手机数据，请：
1. 替换 `data/all_phones_unified.json` 文件
2. 保持JSON格式不变
3. 刷新页面即可看到新数据
